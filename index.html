There are various kind of SDLC models. Some of them are Waterfall, Agile, Iterative, RAD (Rapid Application Development), V Model, Spiral Model.

ring our studies, we learned essential programming concepts, including C programming and data structures like arrays, linked lists, stacks, and queues. We also explored web development with HTML, CSS, and JavaScript, which gave us the skills to create interactive and user-friendly websites.

  • Waterfall: A step-by-step process where each phase (like planning, building, and testing) is completed before moving to the next.
  • Agile: A flexible approach where software is built and delivered in small, working pieces with regular feedback from users.
  • Iterative: Software is developed in small cycles, improving and adding features after each cycle based on feedback.
  • RAD (Rapid Application Development): A fast-paced method to build software quickly using prototypes and frequent user feedback.
  • V-Model: A method where every phase of development has a matching testing phase to ensure quality.
  • Spiral: A process that repeats planning, designing, building, and testing in loops, focusing on reducing risks.

For me, the best model is the Agile model because it is considered the best for modern software development due to its flexibility, adaptability, and customer-focused approach. However deppending on the complexity of project other models can be used.

3 Ways to Include CSS in a Website

1. Inline CSS: CSS is applied directly to an HTML element using the style attribute.

2. Internal CSS: CSS is written within a <style> tag inside the <head> section of the HTML document.

3. External CSS: CSS is written in a separate file with a .css extension and linked to the HTML document using a <link> tag.
